A Windows csoportok szabályzatai

Ezen az oldalon találhatod meg a Windows csoportok szabályzatát.
Kérjük figyelmesen olvasd el az itt leírtakat!

A szabályzat érvényessége

Az alább leírt szabályzat az alábbi csoportokra érvényes:

  • Magyar Windows Mobile Közösség
  • Magyar Windows Közösség
  • Magyar XBOX Közösség
  • Magyar Windows Fejlesztői Közösség
  • Windows Phone Photography
A csoportspecifikus szabályokat külön fejezetben taglaljuk

Adminisztrátorok

Az adminisztrátorok (továbbiakban „admin”) a csoport felelősei. Ők ügyelnek a szabályok betartására, illetve őket lehet megkeresni a csoportokat érintő ügyekben.
